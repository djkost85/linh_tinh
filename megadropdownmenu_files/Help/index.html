A) HTML Structure - top

Please note that in each folder HTML files are the same. This menu has a fixed layout based upon the 960gs grid.

Let's begin with the containers available for the drop downs, you can use from the smallest (130px) to the largest (700px). Then, within these containers, you can organize your content in 1, 2, 3, 4 or 5 columns.

Setting a container

These containers must be placed within a menu list item as follows :

<li> <a href="yourlink">yourlink</a>

    CONTAINER COMES HERE

</li>

And here is the list of the containers available :

To add a 700px drop down :

<div class="dropdown_5columns">

    CONTENT IN 1, 2, 3, 4 OR 5 COLUMNS

</div>

To add a 560px drop down :

<div class="dropdown_4columns">

    CONTENT IN 1, 2, 3 OR 4 COLUMNS

</div>

To add a 420px drop down :

<div class="dropdown_3columns">

    CONTENT IN 1, 2 OR 3 COLUMNS

</div>

To add a 280px drop down :

<div class="dropdown_2columns">

    CONTENT IN 1 OR 2 COLUMNS

</div>

To add a 130px drop down :

<div class="dropdown_1column">

    CONTENT IN 1 COLUMN

</div>

Setting columns for your content

Then, within these containers, you can use from 1 up to 5 columns, but don't forget to check if your container is large enough for the columns you include. You'll notice that they are smaller than the containers, it's because of the paddings so the content doesn't stick to the edges.

Here are the choices available :

<div class="col_1">
    
    HERE COMES YOUR 130 PX WIDTH CONTENT

</div>

<div class="col_2">
    
    HERE COMES YOUR 270 PX WIDTH CONTENT

</div>

<div class="col_3">
    
    HERE COMES YOUR 410 PX WIDTH CONTENT

</div>

<div class="col_4">
    
    HERE COMES YOUR 550 PX WIDTH CONTENT

</div>

<div class="col_5">
    
    HERE COMES YOUR 690 PX WIDTH CONTENT

</div>

B) CSS Files and Structure - top

I'm using 5 CSS files in this theme plus one additional (ie6.css) for the dark variant.

The menu.css is structured, there are labels to help you find the section you need to modify :

	/* 01 RESET */
	some code

	/* 02 PAGE STYLINGS */
	some code
	
	/* 03 MENU BAR */
	some code
	
	/* 04 DROP DOWN STYLINGS */
	some code
	
	/* 05 CONTENT STYLINGS */
	some code
	
	/* 06 CONTACT FORM */
	some code

The menu.css situated in the "all_colors.css" contains stylings for all color variants, specific classes are used in the HTML to separate each style. You may prefer to use a specific color instead of this one.

CSS content

As you can see above, the file is divided in 6 sections :

  • 01 RESET : this is not necessary but I highly recommend you to use a reset at the very beginning of your main CSS file, otherwise, browsers may display the content in an unexpected way (see the sources below for a link with more information).
  • 02 PAGE STYLINGS : in this section you'll find general stylings such as typography, images (borders, floating), general lists stylings.
  • 03 MENU BAR : This section is one of the most importants, it defines the main navigation bar. There are some color stylings and everything which is required for the main navigation.
  • 04 DROP DOWN STYLINGS : this part contains all stylings for the drop downs : show / hide properties, left or right alignment and columns properties (you can use from 1 to 5 columns).
  • 05 CONTENT STYLINGS : this part contains specific stylings for all the content appearing in the drop down : text boxes, lists and icons.
  • 06 CONTACT FORM : I've put here som basic stylings for a simple contact form, please note that this contact form is NOT working, it's here for showing purposes only.

Working with colors

This item comes with 9 color variants, each one has it's own folder, but you may want to create your own color scheme. For this example, we'll see the black variant : open "menu.css" and scroll down until you find the label .menu. This class defines the color of the main menu bar with the following attributes :

	/* Colors */

	background: #333;
	border: 1px solid #444;
	border-top: 1px solid #777;
	
	/* CSS 3 Stylings */
	
	background: -moz-linear-gradient(top, #555, #2a2a2a);
	background: -webkit-gradient(linear, 0% 0%, 0% 100%, from(#555), to(#2a2a2a));

The "background" must be set for browsers which don't support CSS3 properties, in this case, we have the color #333 and this value is overwritten by the CSS3 attributes in the case the user is using a recent browser supporting CSS3, so the 2 last lines define a gradient from the top (color : #555) to the bottom (color : #2a2a2a).

A border is set to match the other colors, as you can see it's a bit lighter (#444) and the top border even more lighter (#777).

All the above properties, as I said before, determine the colors of the navigation bar.

The second thing concerning color are links within the drop down content. These colors are defined in the same file, scroll a little bit down and you'll find the 3 following classes :

  • .menu li:hover div a
  • .menu li:hover div a:hover
  • .menu li ul li a:hover

The first one defines a default link color and the 2 others define the color of the link on mouse hover. So in our black example, the link has the #444 color and becomes #777 on mouse hover.

That's all for the colors specific attributes.
