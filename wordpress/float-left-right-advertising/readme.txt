== Description ==

Float Left Right Advertising can scroll up/down banner when user scroll page up/down. Support HTML, image, flash. Multi options. Thank # and # help me completed this plugin.

Features:

* Can scroll up/down banner when user scroll page up/down 
* Multi options

== Installation ==

1. Download the plugin.
2. Upload it to the plugins folder of your blog.
3. Goto the Settings --> Float Left Right Advertising to config your Ads.
